Vous avez une question? Lisez cette Foire aux Questions et peut être vous trouvez la réponse.

 

  • Quand je suis sur wiggio.com, où est-ce que je dois cliquer pour m’enregistrer?

    Quand vous êtes sur wiggio.com, vous devez cliquer sur ‘New users get started now!’. Vous pouvez trouver ce lien hypertexte en haut de la page au côté droit.

  • Est-il possible d'importer tous mes contacts?

    Oui, c’est possible. Pour importer tout les contacts vous devez cliquer dans la colonne de contacts sur ‘Add’. Puis vous voyez un petit cadre. Cliquez sur ‘Import from email contacts’. Après vous voyez un cadre avec quelques services d’email. Cliquez sur le service que vous voudriez utiliser. Puis vous choisissez vos contacts préférés et vous cliquez sur ‘next’. Finalement vous cliquez sur ‘Add’ et vos contacts sont importés.

  • Est-ce qu’il y a une application pour mon smartphone?

    Non, il n'y a pas d’application pour votre smartphone, mais vous pouvez visiter le site de wiggio avec votre smartphone.

  • De quel navigateur Internet ai-je besoin?

    Vous pouvez utiliser tous les navigateurs (Par exemple: Google Chrome, Safari, Firefox).

  • Est-il possible d’utiliser Wiggio pour des sujets variés?

    Oui, c'est possible, vous pouvez faire différents groupes avec les mêmes personnes. Vous pouvez faire un groupe de français et un groupe de mathématiques par exemple.
